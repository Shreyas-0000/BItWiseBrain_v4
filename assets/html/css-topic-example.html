Introduction to CSS & Types (Inline, Internal, External)

CSS (Cascading Style Sheets) is used to style and layout HTML elements. It separates design from structure, making web development cleaner and more efficient.

Learning Process

  • Inline CSS: Styles applied directly to HTML elements
  • Internal CSS: Styles defined in the <style> tag
  • External CSS: Styles in separate .css files

    • Inline CSS Example:

    <h1 style="color: red;">This is inline CSS</h1>

    Internal CSS Example:

    <head>
  <style>
    h1 { color: blue; }
  </style>
</head>

    External CSS Example:

    /* style.css */
h1 { color: green; }

/* HTML file */
<link rel="stylesheet" href="style.css">

    Real-World Example

    Most modern websites use external CSS files to maintain consistent styling across multiple pages and make updates easier.
