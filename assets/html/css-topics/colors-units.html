Colors, Units & CSS Variables

CSS supports different color values and measurement units. Variables help reuse values throughout stylesheets.

Learning Process

  • CSS supports various color formats (hex, rgb, hsl)
  • Different units for measurements (px, em, rem, %)
  • CSS variables for reusable values

    • CSS Variables:

    :root {
  --main-color: #3498db;
  --text-color: rgb(255, 255, 255);
  --spacing: 2em;
}

    Using Variables:

    body {
  background-color: var(--main-color);
  color: var(--text-color);
  font-size: 16px;
  margin: var(--spacing);
}

    Different Color Formats:

    .element {
  color: #ff0000;        /* Hex */
  color: rgb(255, 0, 0); /* RGB */
  color: hsl(0, 100%, 50%); /* HSL */
}

    Real-World Example

    CSS variables are widely used in modern web design for theming and consistent styling across large applications.
