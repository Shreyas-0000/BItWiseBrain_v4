Introduction to CSS & Types

CSS (Cascading Style Sheets) is used to style and layout HTML elements. It separates design from structure, making web development cleaner and more efficient.

There are three main ways to add CSS to your HTML documents: inline, internal, and external.

Learning Process

  • Inline CSS: Applied directly to HTML elements using the style attribute
  • Internal CSS: Defined within the HTML document using <style> tags
  • External CSS: Stored in separate .css files and linked to HTML documents

    • Inline CSS Example:

    <h1 style="color: blue; font-size: 24px;">Hello World</h1>

    Internal CSS Example:

    <head>
  <style>
    h1 {
      color: blue;
      font-size: 24px;
    }
  </style>
</head>

    External CSS Example:

    /* styles.css */
h1 {
  color: blue;
  font-size: 24px;
}

/* HTML file */
<link rel="stylesheet" href="styles.css">

    Real-World Example

    Most modern websites use external CSS files to maintain consistent styling across multiple pages. This approach makes it easier to update the design of an entire website by modifying a single CSS file.
