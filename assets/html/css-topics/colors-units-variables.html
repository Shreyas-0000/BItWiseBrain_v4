Colors, Units & CSS Variables

CSS provides various ways to specify colors and measurements. CSS Variables (Custom Properties) allow you to store and reuse values throughout your stylesheet.

Understanding different color formats and units is essential for creating consistent and maintainable designs.

Learning Process

  • Color Formats: Hex, RGB, RGBA, HSL, and named colors
  • Units: Absolute (px, pt) and relative (em, rem, %) units
  • CSS Variables: Defining and using custom properties

    • Color Formats:

    /* Hex color */
color: #FF0000;

/* RGB color */
color: rgb(255, 0, 0);

/* RGBA color with opacity */
color: rgba(255, 0, 0, 0.5);

/* HSL color */
color: hsl(0, 100%, 50%);

/* Named color */
color: red;

    Units:

    /* Absolute units */
width: 100px;
font-size: 12pt;

/* Relative units */
width: 50%;
font-size: 1.2em;
padding: 1rem;

    CSS Variables:

    :root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --font-size: 16px;
}

body {
  color: var(--primary-color);
  font-size: var(--font-size);
}

.button {
  background-color: var(--secondary-color);
}

    Real-World Example

    CSS Variables are commonly used in modern web design to create consistent color schemes and maintain design systems. They make it easy to update the entire website's appearance by changing a few values.
