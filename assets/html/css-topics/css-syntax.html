CSS Syntax, Selectors & Specificity

CSS rules consist of a selector and a declaration block. Selectors target HTML elements. Specificity determines which styles apply when multiple rules match.

Learning Process

  • Selectors target HTML elements to apply styles
  • Declaration blocks contain property-value pairs
  • Specificity determines which styles take precedence

    • Basic CSS Rule:

    /* Selector: h1, Property: color, Value: blue */
h1 {
  color: blue;
  font-size: 24px;
}

    Class Selector:

    .text {
  color: red;
}

    ID Selector:

    #main {
  background-color: yellow;
}

    Real-World Example

    Understanding CSS specificity is crucial for maintaining and debugging styles in large projects.
