Introduction to CSS & Types

CSS (Cascading Style Sheets) is used to style and layout HTML elements. It separates design from structure, making web development cleaner and more efficient.

Learning Process

  • Inline CSS: Styles applied directly to HTML elements
  • Internal CSS: Styles defined within the HTML document
  • External CSS: Styles stored in separate files

    • Inline CSS:

    <h1 style="color: red;">This is inline CSS</h1>

    Internal CSS:

    <head>
  <style>
    h1 { 
      color: blue; 
    }
  </style>
</head>

    External CSS:

    /* style.css */
h1 { 
  color: green; 
}

/* HTML file */
<link rel="stylesheet" href="style.css">

    Real-World Example

    Modern web development primarily uses external CSS files to maintain clean separation between structure (HTML) and presentation (CSS), making websites easier to maintain and update.
